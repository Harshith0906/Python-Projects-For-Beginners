## 📌 Overview

This repository is perfect for anyone just starting out with Python. Each project is short, self-contained, and focuses on core programming ideas like **loops**, **conditionals**, **functions**, **randomness**, and **user interaction** — all through fun, hands-on mini-projects.

## 📖 Project Descriptions

### ✂️ 1. Rock Paper Scissors
A classic command-line game where you play against the computer. The computer picks randomly, and the game decides the winner based on standard rules.

**Concepts:** `random`, `if/elif/else`, user input

---

### 🔢 2. Guess the Number
The computer randomly picks a number and gives you hints — "Too High" or "Too Low" — until you guess it correctly.

**Concepts:** `while` loops, `random.randint()`, comparison operators

---

### 🗺️ 3. Choose Your Own Adventure
An interactive text-based story where your choices decide the outcome. Multiple branching paths lead to different endings.

**Concepts:** Functions, nested `if/else`, string input handling

---

### 🎲 4. Lucky Roller
A multiplayer dice-rolling game. Players take turns rolling dice, and the first one to reach a score of 50 wins!

**Concepts:** `while` loops, `random`, turn-based logic, score tracking

---

### 🗺️ 5. Dijkstra's Algorithm Visualization
A beginner-friendly implementation of Dijkstra's shortest path algorithm. Great for understanding how graphs and weighted edges work.

**Concepts:** Graphs, dictionaries, priority queues, algorithm logic

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed → [Download here](https://www.python.org/downloads/)
- No external libraries needed — all standard library!

### Run a Project

**1. Clone the repository**
```bash
git clone https://github.com/Harshith0906/Python-Projects-For-Beginners.git
cd Python-Projects-For-Beginners
```

**2. Run any project**
```bash
python "rock,paper,scissor.py"
python numberguessing.py
python chooseuradvt.py
python Lucky_Roller.py
python pythoncbp.py
```

## 💡 What You'll Learn

- Writing and calling **functions**
- Using **loops** (`for`, `while`) and **conditionals** (`if/elif/else`)
- Generating **random numbers** with the `random` module
- Handling **user input** and input validation
- Implementing a real-world **graph algorithm** (Dijkstra's)
- Structuring small Python programs cleanly

## 🤝 Contributing

Want to add your own beginner project? Contributions are welcome!

1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-project-name`
3. Add your `.py` file with a clear name
4. Update the Projects table in this README
5. Commit and push: `git commit -m 'Add: your project name'`
6. Open a Pull Request

**Guidelines for contributions:**
- Keep it beginner-friendly (no external libraries unless absolutely needed)
- Add comments in your code explaining what each section does
- One project per file
